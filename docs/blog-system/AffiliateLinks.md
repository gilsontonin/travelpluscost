## Auto-discovery script

`node --env-file=.env scripts/viator-opportunities.mjs` scans every post for topics,
queries the Viator Partner API (key: `VIATOR_API_KEY` in `.env`), and writes
`scripts/viator-suggestions.md` with the best-matching product per topic — each URL
already carries your affiliate tracking (`pid`). It flags topics that are already
linked vs. genuinely new. **Read-only; never edits posts.** Always review before
inserting (some free-text matches are off-topic). `scripts/viator-probe.mjs` is a
quick auth/shape check. The generated `viator-suggestions.md` is gitignored.
